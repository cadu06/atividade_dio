Estrutura básica do html

O que é html?

Antes de irmos para a estrutura, precimos conhecer e saber o que html, que se remume a uma linguagem de marcação, html não é linguagem de programação. O nome HTML significa em inglês: HyperText Markup Language. Para gente aqui fica: Linguagem de Marcação de Hipertexto.

Pronto agora podemos ir para a estrutura do html.

O documento HTML sempre inicia com o que chamamos de estrutura básica. Esta estrutura é quase que imutável.

Estrutura básica do html

Tags

O que são tags?

Elas são usadas para estruturar o conteúdo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc. As tags HTML básicas são essenciais para criar qualquer documento HTML.

Exemplos de Tags:
  • < html>: para dizer a esturuta base
  • < head>: como se fosse um cabeçalho para o site
  • < title>: titulo do site
  • < body>: o corpo do site
  • < h1>: tamanho do titulo
  • < p>: paragráfo
  • < del>: parte do texto que foi excluida
  • etc.

Listas: ordenadas e não ordenadas

O que são as listas?

Elas nos ermite criarmos tópicos de textos para uma melhor exemplificação de um determinado assunto. Existem dois tipo listas ordenadas, e não ordenadas, que iremos ver logo abaixo

As listas ordenadas, são as listas que são enumeradas. Ex:

  1. Domingo
  2. Segunda
  3. Terça

As listas não ordenadas, são lsitas que apresentão apenas uma marcação. Ex:

  • Domingo
  • Segunda
  • Terça
